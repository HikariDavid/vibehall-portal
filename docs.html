API REFERENCE

The PolyMind API

PolyMind is a stateless, x402-native intelligence API. Every request is verified on-chain against the x402 protocol, ensuring non-custodial, peer-to-peer data commerce.

Authentication

PolyMind does not use API keys. Authentication is handled via the X-Payment HTTP header. This header must contain a valid x402 payment token proof for the specific endpoint cost.

X-Payment: [x402-token]

Get proofs from the x402 facilitator: x402.autoincentive.xyz

Hot Markets

Fetch the top 20 Polymarket markets by 24h volume. This data is refreshed every 60 seconds from the CLOB WebSocket buffer.

$0.005 USDC GET /markets/hot
Name Type Description
limit integer Max markets to return (default 20, max 50)

Analyze Market

Perform deep AI analysis on a specific market ID. This triggers our multi-model reasoning engine to calculate Fair Value and Edge.

$0.03 USDC GET /analyze/{market_id}
Name Type Description
market_id string The unique Polymarket ID (Required)

Insider Signals

Detect unusual whale activity over the last 1000 trades. Identifies volume clusters that precede price movements.

$0.02 USDC GET /signals

Accuracy Report

View the live Proof-of-Work log. This endpoint is free and allows you to verify our historical performance before paying for signals.

FREE GET /accuracy-report
