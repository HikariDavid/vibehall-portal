03
Autonomous Payments

Since PolyMind is non-custodial, your agent must handle its own x402 payment proof. The agent fetches a payment requirement (402 response), pays via the x402 facilitator, and re-submits the request with the X-Payment header.

Agent Logic Flow
1. Agent calls /analyze
2. PolyMind returns 402 + Payment Details
3. Agent pays USDC via x402 Facilitator
4. Agent retries with X-Payment: [token]
5. PolyMind returns Intelligence JSON
